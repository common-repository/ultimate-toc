Ultimate Table of Contents plugin for specific for content have heading.

== Description ==

A user friendly, featured focused Easy Ultimate Table of Contents plugin which allows you to insert a Ultimate Table of Contents into your default single posts.

<a href="http://toc.neverstopmedia.com/" rel="nofollow ugc">Demo</a>

FEATURES

- Automatically generate a Ultimate Table of Contents for your posts by parsing its contents for headers.
- Supports the Rank Math plugin.
- Works with the Classic Editor, Gutenberg, Divi, Elementor, WPBakery Page Builder and Visual Composer page editors.

Step to plugin-in Ultimate Table of Contents include:

* Install plugin.
* Table of Contents Settings under Settings tab.
* Go to posts show heading before content.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Shortcode will not appear on single page use this shortcode [ultimate_toc]
